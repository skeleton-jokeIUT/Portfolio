Roulette

Accueil

Site réalisé en modèle MVC (voir plus en vidéo).
L'ensemble communique directement avec une base de donnée MySQL via des DAO et DTO.
Ceci est la page par laquelle vous arrivez sur le site. A partir de là vous pouvez vous connecter et jouer à la roulette. Si vous êtes nouveau le module d'inscription est également disponible.

Base de donnée

La base de donnée MySQL est constituée de deux tables.
Une pour les informations du joueurs.
Une pour le suivie des parties (qui joue et est ce qu'il gagne ou perd de l'argent)
